PASO 1️⃣  - Crear Repositorio en GitHub
─────────────────────────────────────────────────────────────────
1. Ve a: https://github.com/new

2. Completa:
   - Repository name: mi-fudo-pro
   - Description: Sistema de Gestión de Pedidos para Restaurantes
   - Visibility: Elige Private o Public (recomendado Private)
   - ❌ NO inicialices con README

3. Click en "Create repository"

4. Copia la URL que aparece (algo como):
   https://github.com/TU_USUARIO/mi-fudo-pro.git


PASO 2️⃣  - Conectar tu código local a GitHub
─────────────────────────────────────────────────────────────────
Abre PowerShell/Terminal en el proyecto y ejecuta:

git remote add origin https://github.com/TU_USUARIO/mi-fudo-pro.git
git branch -M main
git push -u origin main

(Reemplaza TU_USUARIO con tu usuario de GitHub)

Luego verás mensajes como:
  ✓ Enumerating objects
  ✓ Counting objects
  ✓ Writing objects
  ✓ [new branch] main -> main


PASO 3️⃣  - Instalar Wrangler CLI
─────────────────────────────────────────────────────────────────
npm install -g wrangler


PASO 4️⃣  - Autenticarse en Cloudflare
─────────────────────────────────────────────────────────────────
wrangler login

Se abrirá tu navegador. Inicia sesión o crea cuenta (es gratis).


PASO 5️⃣  - Crear Base de Datos D1
─────────────────────────────────────────────────────────────────
wrangler d1 create mi-fudo-pro

📌 IMPORTANTE: Copia el database_id que aparece en la salida
   Ejemplo: "database_id": "xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx"


PASO 6️⃣  - Actualizar wrangler.toml
─────────────────────────────────────────────────────────────────
Abre el archivo wrangler.toml y reemplaza:

database_id = "your-database-id-here"

Por:

database_id = "EL_ID_QUE_COPIASTE_EN_PASO_5"

Ejemplo:
database_id = "12345678-1234-1234-1234-123456789012"


PASO 7️⃣  - Crear las Tablas en D1
─────────────────────────────────────────────────────────────────
wrangler d1 execute mi-fudo-pro --file=./schema.sql

Verás mensajes como "✓ Executed 1 statements"


PASO 8️⃣  - Desplegar a Cloudflare Workers
─────────────────────────────────────────────────────────────────
npm run deploy

O manualmente:

wrangler deploy

Al terminar verás:
✨ Success! Deployed to https://mi-fudo-pro.TU_USUARIO.workers.dev
